## 📚 Exam Features Addressed  

This app completely covers every requirement tested on the standardized exam across all four linguistic skills:

*   **🎧 Listening (25%)**: Hundreds of *Rejoinders* and *Conversations* featuring authentic spoken dialogue.
*   **📖 Reading (25%)**: Reading comprehension covering emails, advertisements, school posters, and contemporary Chinese articles. Contains a seamless toggle between Simplified / Traditional characters.
*   **✍️ Writing (25%)**:  Features the exact visual timer structures. Preprogrammed with **Story Narration** tasks (including 4-panel comic generators) and **Email Reply** prompts.  
*   **🗣️ Speaking (25%)**: Contains browser-based native microphone recording! Simulates the 20-second **Simulated Conversation** recording cutoffs and the **Cultural Presentation** 4m preparation timers.  
*   **💡 Vocabulary & Culture**: Contains a database of almost 1,000 unique AP-level characters, pinyin, and real-world examples derived directly from the CC-CEDICT database.

## ⚙️ Development / Local Setup  

This codebase is meant to be highly accessible and static-hosting friendly (it runs entirely off basic HTML, CSS, and Vanilla JavaScript with no necessary back-end architecture). 

To test it locally yourself:
1. Clone the repository: `git clone https://github.com/lhp620/APChinese.git`
2. Open the project folder.
3. Serve it locally using your preferred HTTP server, such as python or npm:
    ```bash
    npx http-server . -p 8890
    ```
    Or via python: 
    ```bash
    python -m http.server 8890
    ```
4. Find the app running seamlessly at `http://localhost:8890`.
